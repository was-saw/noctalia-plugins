# Launcher Button Plugin

A simple bar widget plugin that provides a button to open the app launcher in Noctalia Shell.

## Features

- **Bar Widget**: Displays a rocket icon button in the bar
- **Quick Access**: Click to toggle the launcher panel
- **Auto-focus**: Automatically clears search text when opening the launcher

## Usage

Click the rocket icon button in the bar to open or close the app launcher panel. The button will toggle the launcher panel on the current screen.

## Requirements

- Noctalia Shell 3.6.0 or higher
